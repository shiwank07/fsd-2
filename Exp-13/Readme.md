# Exp-13: Flask MySQL CRUD API

## Aim
To connect a Flask backend with a MySQL database and perform CRUD operations on student records using SQLAlchemy and Marshmallow validation.

## Technologies Used
- Python
- Flask
- Flask-SQLAlchemy
- Marshmallow
- PyMySQL
- MySQL
