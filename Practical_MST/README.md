# Product Catalog Application

A React-based product catalog application with routing for Dashboard, Products, and Product Details views.

## Features

- Dashboard: Overview of total products and total value
- Products: List of all products with links to details
- Product Details: Detailed view of individual products
- Navigation: Client-side routing with React Router

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:5173](http://localhost:5173) in your browser.

## Build for Production

```bash
npm run build
```
